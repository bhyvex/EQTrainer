SETUP
======
Grant administrative privileges to EQTrainer.exe
Download C++ Redistributable...
x86 - http://www.microsoft.com/en-us/download/details.aspx?id=5555
x64 - http://www.microsoft.com/en-us/download/details.aspx?id=14632


HOTKEYS
=======
CTRL+1 = Teleport to Set 1
CTRL+2 = Teleport to Set 2
CTRL+3 = Teleport to Set 3
CTRL+4 = Teleport to Set 4
CTRL+G = Instant Gate
CTRL+F = Target Follow/unfollow
CTRL+M = Open Map Window


TROUBLESHOOTING
=======
Disable Anti-Virus
Disable UAC (if you have Windows Vista, Windows 7 or Windows 8)
